# Play Store release checklist

## App metadata
- App name
- Short description
- Full description
- App category
- Content rating
- Privacy policy URL

## Graphics
- 512x512 icon
- Feature graphic
- Phone screenshots
- Tablet screenshots

## Technical
- Signed release AAB
- Target API level compatible with Play policy
- App permissions reviewed
- Test on multiple devices

## Store setup
- Google Play Developer account
- App signing key
- Internal testing track for verification
- Production rollout plan
